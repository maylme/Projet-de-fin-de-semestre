************************* COMMENT UTILISER IHM STOCK MANAGER 1.0 **************************
|																														|
|	1.	Placez le dossier GestionDeStock où bon vous semble.											|
|																														|
|	2.	Ouvrez le terminal Linux et rendez-vous dans le dossier GestionDeStock					|
|		EXEMPLE : cd /home/user/Documents/GestionDeStock												|
|																														|
|	3.	Compiler le programme à l'aide de la commande suivante : javac -cp bin src/*/*.java	|
|
|	4.	Lancez le programme à l'aide de la commande suivante : java -cp bin:main main.Main	|
|																														|
*******************************************************************************************



******************************************************* INFORMATIONS UTILES ***************************************************************
|																																														|
| -	Si vous exportez vos données via l'option Export des données, le fichier se trouvera dans le dossier data sous le nom SalleIHM.txt	|
|																																														|
| -	Si vous souhaitez réinitialiser le programme, il suffit de supprimer le contenu du dossier data.												|
|  	ATTENTION, il ne faut pas supprimer le dossier data lui-même !!!
|
|	ATTENTION, il faut le faire fonctionner sous java 1.7 !!				|
|
|	Le mot de passe pour avoir l'autorisation de créer un compte gestionnaire est mdpAdmin.	|																							|
|																																														|
*******************************************************************************************************************************************
